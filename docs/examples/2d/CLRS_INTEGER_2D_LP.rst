CLRS Integer 2D LP
------------------
A 2D LP to motivate fundamental LP concepts and introduce the graphical
method for solving 2-dimensional LPs. See Equations (29.11-29.15) in
*Introduction to Algorithms*, by Cormen, Leiserson, Rivest, and Stein.
